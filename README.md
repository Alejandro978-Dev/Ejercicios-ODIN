# The Odin Project - Fundamentos

Este repositorio contiene los ejercicios y proyectos realizados durante el curso de **Fundamentos de The Odin Project**.  
La meta principal es construir una base sólida en programación y desarrollo web, practicando conceptos esenciales que servirán como punto de partida para proyectos más avanzados.

## Contenido

En este repositorio se encuentran ejercicios organizados por temas como:

- **Introducción a la programación**  
  Conceptos básicos de lógica, estructuras de control y primeros pasos en la resolución de problemas.

- **Git y GitHub**  
  Uso de control de versiones, buenas prácticas y manejo de repositorios.

- **HTML y CSS**  
  Estructuración de páginas web, estilos básicos, selectores y principios de diseño.

- **JavaScript**  
  Sintaxis fundamental, variables, funciones, arreglos, objetos, condicionales y bucles.

- **Prácticas y Proyectos**  
  Pequeños ejercicios aplicados y proyectos iniciales para reforzar el aprendizaje.

## Objetivo del Repositorio

- Documentar el proceso de aprendizaje.  
- Practicar la escritura de código limpio y organizado.  
- Tener una referencia personal del progreso en fundamentos de desarrollo web.  
- Preparar la base para proyectos más grandes en el futuro.

## Tecnologías Usadas

- HTML  
- CSS  
- JavaScript  
- Git & GitHub
